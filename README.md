## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/Kailash1233/Adszoo-Website.git
```

2. Go into project

```bash
cd Adszoo-Website
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
